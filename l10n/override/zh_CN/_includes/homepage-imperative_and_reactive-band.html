统一指令式和响应式

开发应用程序时可以混用熟悉的指令式代码与非阻塞响应式代码.. Learn more.

指令式
@Inject
SayService say;

@GET
@Produces(MediaType.TEXT_PLAIN)
public String hello() {
    return say.hello();
}
响应式（Reactive）
@Channel("kafka") Multi<String> events;

@GET
@Produces(MediaType.SERVER_SENT_EVENTS)
public Multi<String> events() {
    return events;
}
